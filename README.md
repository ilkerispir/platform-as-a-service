# Platform as a service(PaaS) Examples

## [GCP App Engine](https://cloud.google.com/appengine)

### [GCP SDK](https://cloud.google.com/sdk)

###  YAML File
```bash
  runtime: nodejs16
```

###  Deploy Command
```bash
  gcloud app deploy
```
